# Repository Overview

## Purpose

Autonomous Mission Planner demonstrates mission-level autonomy concepts through a modular Python architecture.

The project combines:

- Mission planning
- Route generation
- Risk assessment
- Constraint validation
- Mission simulation
- Human-machine decision support

# Project Scope

## Purpose

Autonomous Mission Planner is a mission-level autonomy simulation framework designed to demonstrate how unmanned systems can evaluate objectives, constraints, threats, and environmental conditions to generate risk-aware mission plans.

The project focuses on planning and decision support rather than vehicle control.

---

## In Scope

The following capabilities are within project scope:

### Mission Planning

- Mission objective evaluation
- Route generation
- Waypoint management
- Mission feasibility assessment

### Risk Assessment

- Threat-zone analysis
- Risk scoring
- Environmental risk evaluation
- Route-level risk assessment

### Constraint Management

- Platform range validation
- Endurance validation
- Mission feasibility checks

### Decision Support

- Mission recommendations
- Replanning recommendations
- Human-readable mission summaries

### Simulation

- Mission scenario execution
- Sample mission datasets
- Route visualization

---

## Out of Scope

The following capabilities are intentionally excluded:

### Vehicle Control

- Flight control
- Navigation control
- Autopilot implementation
- Hardware integration

### Weapon Systems

- Target engagement
- Weapon control
- Fire control systems

### Operational Deployment

- Safety certification
- Flight certification
- Operational mission execution

### Real-Time Command and Control

- Live vehicle communications
- Tactical data links
- Real-time command networks

---

## Intended Audience

This project is intended for:

- Autonomy engineers
- Systems engineers
- Software engineers
- AI/ML practitioners
- Students studying autonomous systems
- Technical recruiters evaluating software projects

---

## Design Philosophy

The project prioritizes:

- Explainability
- Modularity
- Readability
- Operational realism
- Maintainability

over algorithmic complexity.

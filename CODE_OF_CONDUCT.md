# Code of Conduct

## Our Commitment

This project is intended to provide a welcoming, professional, and respectful environment for contributors and users.

Participants are expected to engage in constructive discussion and collaboration.

## Expected Behavior

Examples of positive behavior include:

- Being respectful and professional
- Providing constructive feedback
- Focusing discussions on technical topics
- Supporting a collaborative learning environment
- Respecting differing viewpoints and experiences

## Unacceptable Behavior

Examples of unacceptable behavior include:

- Harassment or personal attacks
- Discriminatory language or conduct
- Trolling or intentionally disruptive behavior
- Publishing private information without permission
- Any conduct that would be considered unprofessional in a workplace environment

## Scope

This Code of Conduct applies to:

- Repository discussions
- Pull requests
- Issues
- Project documentation
- Community interactions related to the project

## Enforcement

Project maintainers reserve the right to remove, edit, or reject comments, commits, code, documentation, issues, or other contributions that violate this Code of Conduct.

## Attribution

This Code of Conduct is adapted from common open-source community standards and is intended to support a professional engineering environment.

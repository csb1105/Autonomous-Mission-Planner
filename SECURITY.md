# Security Policy

## Supported Versions

This repository is a demonstration and educational project.

Security updates may be provided periodically, but no formal support guarantees are offered.

| Version | Supported |
|----------|----------|
| 0.1.x | Yes |
| < 0.1 | No |

## Reporting a Vulnerability

If you identify a security issue within this repository:

1. Do not publicly disclose the issue immediately.
2. Document the vulnerability and its potential impact.
3. Open a private communication channel with the repository maintainer when possible.
4. Allow reasonable time for investigation and remediation.

## Scope

This repository is intended for:

- Educational use
- Portfolio demonstrations
- Software architecture examples
- Mission-planning simulations

It is not intended for operational deployment, safety-critical applications, or real-world vehicle control.

The software contained in this repository is provided as-is without warranty of any kind.

Users are responsible for evaluating suitability before use in any environment.
